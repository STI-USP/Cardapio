# Checklist — Pronto para Lançar

- [ ] Smoke/regressão mínima passou (sem crashes, sem spinners >10s).
- [ ] Testes automatizados críticos passaram (unit/integration/UI relevantes).
- [ ] Métricas chave revisadas: sucesso/erro, latência p95 dos fluxos tocados.
- [ ] Regressão mínima executada para áreas impactadas; bugs críticos/altos zerados.
- [ ] Rollback/feature flags definidos (se aplicável) ou plano de reversão documentado.
- [ ] Config vars confirmadas para ambiente de release (URLs, tokens, secrets).
- [ ] Notas de release e riscos conhecidos documentados.
- [ ] Evidências anexadas (logs, relatórios, screenshots quando UI relevante).
